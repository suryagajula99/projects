# Live Portfolio Analyzer using Python

## Overview

This project automates portfolio analysis by reading stock holdings from an Excel file, retrieving live stock prices from Yahoo Finance, calculating the portfolio performance, and storing the results in an Oracle database.

## Features

- Reads stock holdings from Excel
- Retrieves live stock prices using Yahoo Finance
- Calculates:
  - Current Price
  - Previous Close
  - Daily Change (%)
  - Portfolio Contribution
- Inserts results into Oracle Database
- Exports the updated portfolio to Excel

## Technologies

- Python
- Pandas
- yfinance
- Oracle Database
- openpyxl
